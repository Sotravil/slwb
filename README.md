## Instala Termux para proseguir
>playstore (android) : https://play.google.com/store/apps/details?id=com.termux&hl=en
>appstore (iphone) : https://apps.apple.com/gr/app/termius-ssh-client/id549039908

## Clonar Este Repositorio

```bash
> git clone https://github.com/Sotravil/slwb
```

## Instalar las dependencias:
Antes de ejecutar el siguiente comando, asegúrese de estar en el directorio del proyecto que
acabas de clonar !!

```bash
> cd slwb
> bash install.sh
```

### Uso
```bash
> npm start
```
